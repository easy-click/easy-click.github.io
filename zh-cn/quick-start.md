* quick start

> ```java
> String x = "string";
> String y = "string";
> String z = new String("string");
> 
> System.out.println("x == y:"+(x == y)); //true
> System.out.println("x == y:"+(x == z)); //false
> 
> System.out.println("x.equals(y):"+x.equals(y)); // true
> System.out.println("x.equals(z):"+x.equals(z)); // true
> ```
