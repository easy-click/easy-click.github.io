1. 说明

  • 设备模块函数主要是跟设备信息相关联
  • 设备模块的对象前缀是device，例如 device.getIMEI()这样调用

2. 载入模块

loadECModule("device");

3. 函数

3.1. device.getIMEI()

  • 获取手机的IMEI
  • @return 字符串
var imei = device.getImei();
toast(imei);

3.2. device.getScreenWidth()

  • 取得屏幕宽度
  • @return 整型
var width = device.getScreenWidth();
toast(width);

3.3. device.getScreenHeight()

  • 取得屏幕高度
  • @return 整型
var height = device.getScreenHeight();
toast(height);

3.4. device.getAndroidId()

  • 取得Android ID
  • @return 字符串
var androidId = device.getAndroidId();
toast(androidId);

3.5. device.getBrand()

  • 取得手机品牌
  • @return 字符串
var brand = device.getBrand();
toast(brand);

3.6. device.getModel()

  • 取得手机机型
  • @return 字符串
var model = device.getModel();
toast(model);

3.7. device.getImsi()

  • 取得手机卡号
  • @return 字符串
var imsi = device.getImsi();
toast(imsi);

3.8. device.getSerial()

  • 取得手机串号
  • @return 字符串
var serial = device.getSerial();
toast(serial);

3.9. device.getSdkInt()

  • 取得手机SDK 版本号，例如 23
  • @return 字符串
var sdkInt = device.getSdkInt();
toast(sdkInt);

3.10. device.getOSVersion()

  • 取得手机版本号,例如 6.0等字符串
  • @return 字符串
var osVersion = device.getOSVersion();
toast(osVersion);

3.11. device.getMacAddress()

  • 获取mac地址
  • @return 字符串
var res = device.getMacAddress();
toast(res);

3.12. device.getBattery()

  • 取得电量
  • @return int型
var res = device.getBattery();
toast(res);

3.13. device.getTotalMem()

  • 取得总内存
  • @return long型
var res = device.getTotalMem();
toast(res);

3.14. device.getAvailMem()

  • 取得可用内存
  • @return long型
var res = device.getAvailMem();
toast(res);

3.15. device.isCharging()

  • 是否正在充电
  • @return 布尔型
var res = device.isCharging();
toast(res);

3.16. device.vibrate()

  • 发出震动 单位是毫秒
     device.vibrate(1*1000);

3.17. device.cancelVibration()

  • 取消震动
     device.cancelVibration();
